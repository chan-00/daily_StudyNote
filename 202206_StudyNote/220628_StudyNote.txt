1. 생활코딩 문서 공부
	> WEB1 - CSS 내용 정리
		1) HTML 이후 디자인만을 위한 언어의 필요성 대두 -> CSS가 나타나게 된 계기

		2) 주석 처리는 <!-- 내용 --> 으로 함

		3) CSS 언어는 head 태그 안의 style 태그 안에다가 작성해야 함

		4) CSS 사용 이유?
		-> 코드 수정 용이, 관리 편함, 코드 중복의 제거(유지 보수가 쉽고, 가독성이 높아짐)

		5) CSS에서 효과 지정 후 사이에 세미콜론(;)으로 구분해 줘야 한다.

		6) html에서 속성 중 style 속성을 사용하면 css 효과를 넣을 수 있다.

		7) CSS 선택자
		태그 선택자, 클래스 선택자, id 선택자가 있으며 여러 CSS 선택자의 효과가 적용된다고 하면 id, 클래스, 태그 선택자 순으로
		우선순위에 따라 CSS 효과가 적용된다.

		8) CSS에서의 주석은 /* 내용 */ 으로 표현한다.

		9) 태그 중 한 줄을 모두 차지하는 태그는 block, 컨텐츠의 영역만큼의 크기만 차지하는 태그를 inline 태그라 한다.
		또한 padding은 컨텐츠와 태그의 영역 사이의 간격을 의미하고 margin은 다른 태그들과의 간격을 의미한다.

		10) display: grid를 사용하면 해당 컨테이너의 자식 태그 영역의 배치를 할 수 있다.

		11) link 태그를 사용하여 CSS 파일을 불러와 적용하면 수정 사항이 생겼을 때 CSS 파일 하나만 수정하면 해당 CSS 파일이 적용된 모든
		페이지가 수정되는 효과를 얻을 수 있다.

		+) CSS 속성들
			- color: 색상;
			태그 안의 내용들의 색상을 지정함

			- text-decoration: none
			텍스트의 장식(a 태그의 밑줄 등)을 없앤다.
