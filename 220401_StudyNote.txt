2. 학교 강의 필기
	<알고리즘 강의>
	- 정렬 Sort : 주어진 데이터를 일정 규칙에 따라 다시 나열(오름차순 또는 내림차순)
	
	- 버블 정렬(간단하지만 성능 낮음)
		1) 인접한 2개씩 비교
		2) 한 번 정렬할 때 맨 마지막 원소 결정

	- 버블정렬 특징
		1) 안정적 알고리즘, 제자리 알고리즘(입력공간 이외 별도 메모리X)
		2) 최악의 경우 O(n의 2승)

	- 삽입 정렬 : 미정렬 부분의 첫 번째 원소로부터 정렬된 부분의 제자리에 삽입
		-> 최악의 경우 O(n의 2승)

	- 선택 정렬(자리 교환이 위 두 정렬보다 적게 일어남)
		1) 미정렬 데이터 중 가장 작은 것 선택, 그 데이터를 미정렬 데이터 첫 번째 원소와 교환하는 과정 반복
		2) 최악의 경우 O(n의 2승)
		3) 안정적/불안정적 알고리즘(한 번 위치가 정해진 후에도 위치가 바뀔 수 있다.)



	<운영체제 실습 강의>
	- vi에서 입력에 관련된 키
		1) i 키 : 현재 커서에 입력, a 키 : 현재 커서의 오른쪽에 입력
		2) s 키 : 현재 커서의 문자 지워짐, o 키 : 다음 줄에 입력
		3) shift + i 키 : 현재 라인 맨 앞에 입력, shift + a 키 : 현재 커서 라인의 마지막 위치에 입력
		4) shift + s 키 : 현재 커서 라인의 내용 모두 삭제, shift + o 키 : 현재 커서 위에 한 줄 만듦
		
	- vi에서의 커서 이동 키
		1) 커서 왼쪽, 아래, 위, 오른쪽 이동(순서대로 h, j, k, l 키)
		2) shift + 4번키 : 현재 라인 맨 뒤로 커서 이동, shift + 6번키 : 현재 라인 맨 앞으로 이동
		3) gg 키 : 현재 문서 맨 앞 문자로 이동, shift + g 키 : 현재 문서 맨 밑줄 문자로 이동
		4) (줄 숫자)+(shift+g) 키 : 해당 라인으로 이동, :(줄 숫자) : 해당 라인으로 이동
		
	- vi에서 문자 삭제 키
		1) x 키 : del 키와 같음, 현재 커서의 문자 지움
		2) shift + x 키 : backspace 키와 동일
	
	- vi에서 복사 + 붙여넣기 키
		1) yy 키 : 현재 줄 복사, (숫자)yy 키 : 현재 라인부터 해당 숫자만큼 밑 라인 복사
		2) p 키 : 현재 커서라인 밑줄에 붙여넣기, shift + p 키 : 현재 커서라인 줄에 붙여넣기

	- vi에서 라인 지우기
		1) dd 키 : 현재 커서 라인 지우기, (숫자)dd 키 : 숫자 만큼의 라인 지우기
	
	- vi에서 문자 찾기 관련 키
		1) :/(찾는 문자열) => 해당 문자열 찾음
		2) :%s/(찾는 문자열)/(바꿀 문자열) => 해당 문자열을 찾아 뒤의 문자열로 바꿈.

	- 디렉토리 생성(mkdir (디렉토리명))

	- 위치 지정(cd)
		1) 절대 경로(/~)
			현재 위치에서 root 디렉토리를 기준으로 경로를 찾아감
		2) 상대 경로(../)
			현재 디렉토리 기준 경로

	- 파일 복사
		1) cp 명령어
		cp (복사할 파일) (복사할 경로)
		ex) cp text.txt ../directory 
		> (현 디렉토리의 text.txt란 파일을 현재의 상위 디렉토리의 아래에 있는 directory 디렉토리에 복사하라는 뜻)
